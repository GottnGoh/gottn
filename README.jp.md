# gottn

- Gottnは、ユーザーインターフェイスを構築するためのJavaScriptライブラリです。
- jQueryやnode.jsなどの他のライブラリは必要ありません。
- これだけがあれば動きます。
- とてもシンプルで覚える事は少ないです。

# インストール

```html
	<script type='text/javascript' src='js/gottn.min.js'></script>
```
